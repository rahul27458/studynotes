# [StudyNotes.org](http://studynotes.org)

Now with 100% more open source goodness!

## Installation

### Server dependencies
- Node.js
- Nginx
- MongoDB
- Cairo
  - OS X: Install XQuartz, then `brew install cairo`
  - Ubuntu: `sudo aptitude install libcairo2-dev`
